## mongdb

# 基础操作
```js
mongo
help
exit
mongo
show dbs
use komablog
show collections
db.createCollection('user')
show collections
show dbs
db.stats()
db.dropDatabase()
show dbs
```
# 操作集合
```js
mongo
show dbs
show collections
db.createCollection('user')
show collections
db.users.crnameCollection('users')
show collections()
db.staff.drop()
show collections
db.dropDatabase()
show dbs
```
